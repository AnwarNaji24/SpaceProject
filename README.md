## Description
This project is a space-themed web application.  
It includes user registration and login functionality.  

The project provides information and facts about space and planets.  
It also includes a quiz that allows users to test their knowledge about planets.  

The system includes a profile page that displays:
- Username  
- Email  
- Number of quizzes taken  
- Score of the last quiz  

(Note: The password is stored securely and is not displayed.)

## Required Environment
- HTML  
- CSS  
- PHP  
- MySQL  
- Apache  
- XAMPP  
- NetBeans
